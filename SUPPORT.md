# Support

## Getting help

### Bug reports and feature requests

Open a [GitHub Issue](https://github.com/Dev-Community-IO/devcommunityIO/issues) using the appropriate template:

- **Bug report** — something is broken
- **Feature request** — propose new functionality

### Security issues

Do **not** use public issues for security reports. See [SECURITY.md](SECURITY.md).

### Documentation

- [README](README.md) — installation and development setup
- [CONTRIBUTING.md](CONTRIBUTING.md) — how to contribute
- [Backend API repo](https://github.com/Dev-Community-IO/devcommunityV2-api) — required for full-stack local development

### Community

- Website: [https://devcommunity.io](https://devcommunity.io)
- GitHub Discussions (if enabled on the repository)

## Commercial support

This is an open-source community project. Enterprise or hosted support is not
officially provided through this repository. Sponsorship options may be available
via [GitHub Sponsors](https://github.com/sponsors/Dev-Community-IO).

## Response expectations

Maintainers are volunteers. Issues are triaged as time permits. Providing clear
reproduction steps and environment details helps us respond faster.
